# Backend - Telegram Bot Builder

Node.js сервер для управления Telegram ботами.

## Установка

```bash
npm install
```

## Запуск

### Production
```bash
npm start
```

### Development (с автоперезагрузкой)
```bash
npm run dev
```

Сервер запустится на порту 5000.

## Переменные окружения

Создайте файл `.env` (опционально):

```env
PORT=5000
```

## Архитектура

### server.js
Главный файл Express сервера с API endpoints.

### bot-manager.js
Класс для управления жизненным циклом ботов:
- Добавление/удаление ботов
- Запуск/остановка ботов
- Хранение конфигураций

### code-generator.js
Генерирует исполняемый код бота на основе блоков:
- Поддерживает различные типы блоков
- Создает обработчики событий
- Возвращает функцию для инициализации бота

## Блоки

Каждый блок имеет:
- `type` - тип блока (start, command, message и т.д.)
- `data` - данные блока (текст, команда, параметры)
